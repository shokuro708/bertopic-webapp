# README.md
# BERTopic Web分析ツール

Streamlitを使用したBERTopicベースのテキスト分析Webアプリケーション

## 機能
- CSVファイルからのテキスト分析
- トピックモデリング
- インタラクティブな可視化
- 結果のダウンロード

## ローカル実行方法
```bash
pip install -r requirements.txt
streamlit run app.py
```

## デプロイ方法
1. GitHubにプッシュ
2. Streamlit Cloudでアプリを作成
3. 自動デプロイ
